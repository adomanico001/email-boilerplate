# Responsive Email Boilerplate
A responsive, modular email framework built for marketing campaign systems.

This project demonstrates a fully responsive, Outlook-safe HTML email coded from scratch. <br>
Built using table-based layouts, inline CSS, VML buttons, and accessibility best practices.

## Features
- 600px modular container
- Dark mode support
- Bulletproof buttons for Outlook
- Mobile-first responsive design
- UTM-tagged CTA tracking
- Tested in Gmail, Outlook, and Apple Mail
